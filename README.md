# Superstore 
This project consist of a front-end written in React,js, a back-end written in c#, and MS SQL stored procedures.

## Superstore API 
---
### Product
- **GET** /api/products
- **GET** /api/products/{id}
- **POST** /api/products
- **PUT** /api/products/{id}
- **DELETE** /api/products/{id}
---
